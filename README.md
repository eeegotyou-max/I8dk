## Usage

You **cannot** deploy Waves on any static hosting services!

## Self-Hosting
   
```bash
git clone https://github.com/l4uy/waves/

cd waves

sh setup.sh
```

## Running locally

```bash
git clone https://github.com/l4uy/waves/

cd waves

bun i

bun dev
```
